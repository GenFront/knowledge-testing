Алгоритм разметки ⏳
  1. Используем базовый шаблон разметки.
  2. Размечаем крупные структурные блоки страниц.
  3. Внутри структурных блоков выделяем смысловые разделы.
  4. Выделяем заголовок всего документа и заголовки смысловых разделов.
  5. Внутри смысловых разделов размечаем мелкие элементы, пытаясь подобрать подходящие по смыслу теги.
  6. Мелкие элементы, для которых не получается подобрать тег, размечаем с помощью универсальных тегов.

На каждом шаге, с 2 по 6, подбираем и добавляем классы.

Emmet Команды ⏳
  • Дочерние элементы создаются так: nav>ul>li
  • Соседние элементы создаются так: div+p+bq
  • Выйти на один уровень выше можно так: div+div>p>span+em^bq
  • Выйти на два уровня выше можно так: div+div>p>span+em^^bq
  • a href="#" - a[href]
  • Несколько элементов создаются так: ul>li*3
  • Можно использовать символ $ для того чтобы задавать номер в названии класса: ul>li.item$*2
  • Или таким образом:
    • ul>li.item$$$*3
    • ul>li.item$@-*3
    • ul>li.item$@3*5
  • Группировка делается с помощью круглых скобок: div>(header>ul>li*2>a)+footer>p
  • Классы и идентификаторы создаются так:
    • #header (идентификатор): 
      <div id="header"></div>
    • .title (класс): 
      <div class="title"></div>
    • form#search.wide (комбинированный селектор): 
      <form id="search" class="wide"></form>
  • Произвольные атрибуты создаются так:
    • p[title="Hello world"] 
      <p title="Hello world"></p>
    • td[rowspan=2 colspan=3 title] 
      <td rowspan="2" colspan="3" title=""></td>
    • [a=value1 b=value2] 
      <div a="value1" b="value2"></div>
  • Чтобы указать текст нужно заключить его в фигурные скобки, так:
    • a{Click me} 
      <a href="">Click me</a>
    • p>{Click }+a{here}+{ to continue} 
      <p>Click <a href="">here</a> to continue</p>
