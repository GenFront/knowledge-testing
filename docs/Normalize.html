Normalize.css - Руководство использования

Что это?

Normalize.css - это CSS-файл, который обеспечивает единообразное отображение HTML-элементов в разных браузерах.

Установка

Файл устанавливаем в папку: style/normalize.css

Создаем структуру папок вручную:

  • Создаем папку style: 
    # В терминале, находясь в папке проекта:
    mkdir style
  • Теперь скачиваем normalize: 
    
    curl -o style/normalize.css https://cdnjs.cloudflare.com/ajax/libs/normalize/8.0.1/normalize.min.css

Подключение

<head>
    <!-- Normalize должен подключаться ПЕРВЫМ -->
    <link rel="stylesheet" href="../style/normalize.css">
    <!-- Затем остальные стили -->
    <link rel="stylesheet" href="style.css">
</head>

Что делает Normalize?

  • Убирает margin/padding по умолчанию
  • Стандартизирует размеры заголовков
  • Исправляет баги браузеров
  • Обеспечивает единый line-height

Преимущества

  • Кроссбраузерная совместимость
  • Сохранение полезных стилей по умолчанию
  • Легковесность (всего 1.5KB)

Частые вопросы

Почему margin: 0 перечеркнут в DevTools?
A: Это нормально! Normalize устанавливает базовые стили, которые вы можете переопределять.

Q: Нужно ли удалять Normalize после разработки?
A: НЕТ! Normalize должен оставаться в продакшене навсегда.
