Структура React-компонента

React-компонент — это независимая часть пользовательского интерфейса, которая может иметь собственное состояние и жизненный цикл.

  }

Жизненный цикл компонента

  • Монтирование — создание и вставка компонента в DOM
  • Обновление — изменение состояния или пропсов
  • Размонтирование — удаление компонента из DOM

Хуки в функциональных компонентах

  • useState — для управления состоянием
  • useEffect — для побочных эффектов
  • useContext — для работы с контекстом
  • useReducer — альтернатива useState для сложного состояния
  • useCallback и useMemo — для оптимизации
