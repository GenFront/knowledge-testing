Шпаргалка: Grid vs Flex — Cheatsheet

Коротко и по делу: куда применять Grid, куда — Flex. С готовыми паттернами и ошибками, которые лучше не повторять.

1 — Общий принцип

Grid — для структуры страницы (layout). Flex — для внутренних элементов (micro-layout).

2 — Когда использовать GRID

Подходит для каркаса страницы, галерей, dashboard, двумерных структур.

  • Страница: header / main / sidebar / footer
  • Галереи и каталоги
  • Двумерные структуры — строки и колонки

3 — Когда использовать FLEX

Идеально для кнопок, меню, выравнивания вдоль одной оси.

  • Кнопки с иконками
  • Header: логотип + меню
  • Вложенные элементы карточки

4 — Правило идеальной комбинации

Layout = GRID. Micro-layout = FLEX.

6 — Типичные ошибки

  • Страницу строят на Flex с кучей вложенностей — плохо для адаптивности.
  • Использовать Grid для простой строки — избыточно.
  • Использовать float — устарело: Grid/Flex закрывают кейсы float.
  • Grid без gap — трудоёмкие костыли с margin.
  • Перебор с медиа-запросами когда `auto-fit`/`minmax` решают задачу.

7 — Практические паттерны

Галерея карточек (Grid) 
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: 20px;
}
Header (Flex) 
header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 20px;
}
Кнопка с иконкой 
.button {
  display: flex;
  align-items: center;
  gap: 10px;
}
Двумерная сетка для dashboard 
.dashboard {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 24px;
}
.widget-large { grid-column: span 6 }
.widget-small { grid-column: span 3 }
Вертикальная карточка (Flex-column) 
.card {
  display: flex;
  flex-direction: column;
  gap: 12px;
}
Макет страницы (Grid + Flex внутри) 
.page {
  display: grid;
  grid-template-rows: auto 1fr auto;
  grid-template-columns: 250px 1fr;
  min-height: 100vh;
}
header { grid-column: 1 / -1; display:flex; justify-content:space-between; padding:20px }
.sidebar { grid-row:2 }
.main { grid-row:2; display:grid; gap:20px; grid-template-columns:repeat(auto-fill, minmax(280px, 1fr)) }
