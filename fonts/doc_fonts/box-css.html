Модель CSS Box

Что такое CSS Box Model

Модель CSS Box — это концепция, описывающая, как каждый элемент на веб-странице представлен в виде прямоугольной коробки.

Компоненты CSS Box Model

  • Content — само содержимое элемента (текст, изображения и т.д.)
  • Padding — внутренние отступы между содержимым и границей
  • Border — граница вокруг содержимого и отступов
  • Margin — внешние отступы между границей и другими элементами

Свойства, связанные с Box Model

  • width и height — ширина и высота содержимого
  • padding — внутренние отступы
  • border — границы
  • margin — внешние отступы
  • box-sizing — как рассчитываются ширина и высота элемента

Значения box-sizing

  • content-box — ширина и высота применяются только к содержимому (по умолчанию)
  • border-box — ширина и высота включают содержимое, padding и border (но не margin)
