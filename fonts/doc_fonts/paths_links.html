Пути и ссылки

Типы путей в веб-разработке

Относительные пути

Относительные пути указывают местоположение файла относительно текущего файла:

  • filename.html — файл в той же папке
  • ../folder/filename.html — подняться на уровень вверх, затем перейти в папку folder
  • ./folder/filename.html — файл в папке folder (в той же директории)
  • ../../folder/filename.html — подняться на два уровня вверх, затем перейти в папку folder

Абсолютные пути

Абсолютные пути указывают полный путь к файлу:

  • /folder/filename.html — путь от корня сайта
  • https://example.com/folder/filename.html — полный URL-адрес

Типы ссылок

  • Внутренние — ссылаются на другие страницы в том же сайте
  • Внешние — ссылаются на другие сайты
  • Якорные — ссылаются на определенную часть той же страницы
  • Скачивание файлов — ссылаются на файлы для скачивания

Атрибуты ссылок

  • href — адрес ссылки
  • target — где открыть ссылку (_self, _blank, _parent, _top)
  • rel — отношение между текущим документом и документом ссылки (например, nofollow, noopener)
  • title — всплывающая подсказка

Лучшие практики

  • Использовать осмысленный текст для ссылок
  • Открывать внешние ссылки в новой вкладке с rel="noopener noreferrer"
  • Проверять, что все ссылки работают корректно
  • Использовать абсолютные пути от корня для внутренних ссылок в больших проектах
