CSS — высоты, flex, grid и CSS-переменные

Работа с высотами в CSS

В CSS есть различные способы управления высотой элементов:

  • height — фиксированная высота
  • min-height — минимальная высота
  • max-height — максимальная высота
  • 100vh — высота, равная высоте окна просмотра

Flexbox

Flexbox — это одномерная система макета, которая позволяет легко распределять пространство между элементами в контейнере.

Основные свойства: display: flex, flex-direction, justify-content, align-items.

Grid

CSS Grid — это двумерная система макета, которая позволяет создавать сложные макеты с помощью строк и столбцов.

Основные свойства: display: grid, grid-template-columns, grid-template-rows, gap.

CSS-переменные

CSS-переменные (кастомные свойства) позволяют хранить значения, которые можно повторно использовать в разных местах.

Пример: :root { --primary-color: #007bff; } и использование color: var(--primary-color);
