Классическая схема верстки

Что это за схема?

  1. Ограничивающий контейнер 
    
  <div class="container">
  <!-- Весь контент сайта -->
  </div>

    Назначение: Контроль максимальной ширины, центрирование, общие отступы.

  2. "Обнуленный" CSS (Reset/Normalize) 
    
  /* Обнуление стандартных стилей браузера */
  * { margin: 0; padding: 0; box-sizing: border-box; }
  ul { list-style: none; }
  a { text-decoration: none; color: inherit; }
  /* и т.д. */

Зачем это нужно?

Проблема: Браузеры имеют встроенные стили

  • Разные отступы у <body>, <h1> - <h6>, <p>, <ul></ul>
  • Разные шрифты и их размеры
  • Разные цвета ссылок

Решение: Reset CSS приводит все к единому виду

Какие селекторы обнуляются по умолчанию:

Базовые обнуления:


  /* 1. Box model */
*, *::before, *::after {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

/* 2. Списки */
ul, ol {
  list-style: none;
}

/* 3. Ссылки */
a {
  text-decoration: none;
  color: inherit;
}

/* 4. Кнопки */
button {
  border: none;
  background: none;
  cursor: pointer;
}

/* 5. Изображения */
img {
  max-width: 100%;
  height: auto;
  display: block;
}

/* 6. Формы */
input, textarea, select {
  border: none;
  outline: none;
  font-family: inherit;
}

Почему именно эти селекторы:

  1. Box-model - чтобы отступы были предсказуемыми
  2. Списки - убирают маркеры для навигаций
  3. Ссылки - убирают подчеркивание для дизайна
  4. Кнопки - убирают стандартные стили ОС
  5. Изображения - предотвращают растягивание
  6. Формы - единообразие across browsers

Популярные подходы:

Reset CSS (полное обнуление):


  * { margin: 0; padding: 0; }

Normalize.css (щадящее выравнивание):


  /* Сохраняет полезные стили, но приводит к единому виду */

Modern approach (компромисс):


  :where(*) {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Преимущества схемы:

  1. Единообразие во всех браузерах
  2. Предсказуемость отступов и размеров
  3. Чистая основа для собственных стилей
  4. Кроссбраузерность с меньшими усилиями

Эта схема — профессиональный стандарт для предсказуемой верстки!
