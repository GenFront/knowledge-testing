Debug CSS — шпаргалка

Как использовать Super Debug Extended

Чтобы быстро видеть структуру DOM, размеры блоков и flex/grid контейнеры, следуй этим шагам:

  1. Подключи CSS к своему проекту: 
    <link rel="stylesheet" href="super-debug-extended.css">
  2. Добавь кнопку Debug в body для включения и выключения: 
    <button id="debug-toggle">🛠 Debug</button>

    При нажатии кнопки класс debug добавляется на <body> и подсветка включается.

  3. Подключи JS для кнопки и горячей клавиши: 
    <script>
const toggle = document.getElementById("debug-toggle");
toggle.addEventListener("click", () => document.body.classList.toggle("debug"));

document.addEventListener("keydown", e => {
  if (e.key === "d" && e.ctrlKey) {
    document.body.classList.toggle("debug");
  }
});
</script>

    Теперь Ctrl+D также включает/выключает debug.

Что ты видишь при включении

  • Все блоки страницы подсвечены outline разного цвета по уровням вложенности
  • Flex и Grid контейнеры выделены пунктирной рамкой
  • На каждом элементе видны: W, H, padding, margin, display, классы и id
  • Элемент под курсором подсвечивается яркой рамкой

Как использовать для практики

  1. Открой любой HTML проект и подключи этот CSS.
  2. Включи debug кнопкой или Ctrl+D.
  3. Наведи курсор на блоки, чтобы увидеть их размеры и свойства.
  4. Изучи структуру DOM, flex/grid, padding и margin.
  5. Копируй блоки кода, классы или стили в свои шпаргалки.

Этот файл можно использовать как шпаргалку по визуализации верстки в твоей библиотеке: открываешь, включаешь debug и копируешь нужные куски к себе.
