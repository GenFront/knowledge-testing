UI / UX — практический справочник

UI (User Interface) - пользовательский интерфейс

UI — это визуальная часть продукта, с которой взаимодействует пользователь: кнопки, формы, иконки, цвета, типографика и т.д.

Принципы UI-дизайна

  • Единообразие — одинаковые элементы должны выглядеть и вести себя одинаково
  • Четкость — интерфейс должен быть понятным и интуитивно понятным
  • Контрастность — важные элементы должны выделяться
  • Доступность — интерфейс должен быть доступен для всех пользователей

UX (User Experience) - пользовательский опыт

UX — это общий опыт пользователя при взаимодействии с продуктом: удобство, эффективность, удовлетворенность и т.д.

Принципы UX-дизайна

  • Понятность — пользователь должен понимать, что происходит на сайте
  • Доступность — продукт должен быть доступен для всех пользователей
  • Эффективность — пользователь должен быстро достигать своих целей
  • Удовлетворенность — взаимодействие должно приносить удовольствие

Этапы UX-дизайна

  1. Исследование (research)
  2. Анализ (analysis)
  3. Проектирование (design)
  4. Тестирование (testing)
  5. Реализация (implementation)

Инструменты UI/UX

  • Figma — для проектирования интерфейсов
  • Adobe XD — альтернатива Figma
  • Sketch — популярный инструмент для macOS
  • InVision — для прототипирования
  • Zeplin — для передачи макетов разработчикам
