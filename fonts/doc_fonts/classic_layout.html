Классическая схема верстки

Классическая схема верстки с reset/normalize CSS

Профессиональный стандарт для предсказуемой верстки, обеспечивающий единообразное отображение элементов в разных браузерах.

Что включает в себя классическая схема:

  • Reset или Normalize CSS для сброса/нормализации стилей браузера по умолчанию
  • Единый подход к отображению элементов в разных браузерах
  • Предсказуемое поведение элементов при изменении условий
  • Базовая структура документа
  • Система сеток для адаптивной верстки
  • Модульный подход к CSS

Reset vs Normalize

Reset CSS — полностью сбрасывает все стили браузера по умолчанию, устанавливая одинаковые значения для всех элементов.

Normalize CSS — нормализует стили, сохраняя полезные значения по умолчанию и устраняя различия между браузерами.

Пример структуры HTML документа

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Название страницы</title>
    <link rel="stylesheet" href="css/normalize.css">
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    <header class="page-header">
        <!-- Шапка сайта -->
    </header>
    
    <main class="page-main">
        <!-- Основное содержимое -->
    </main>
    
    <footer class="page-footer">
        <!-- Подвал сайта -->
    </footer>
</body>
</html>

Преимущества классической схемы

  • Предсказуемость поведения элементов
  • Единообразие отображения в разных браузерах
  • Легкость в поддержке и развитии
  • Совместимость с различными CSS-фреймворками
  • Удобство для командной разработки
