Чек-лист

Чек-лист для проверки готового сайта

HTML

  • ✓ Правильная структура документа (DOCTYPE, html, head, body)
  • ✓ Семантическая разметка (header, nav, main, article, section, footer и т.д.)
  • ✓ Правильное использование заголовков (h1-h6) по иерархии
  • ✓ Все изображения имеют атрибут alt
  • ✓ Все ссылки имеют осмысленный текст
  • ✓ Формы имеют правильные метки (label)
  • ✓ Код валиден (проверить на validator.w3.org)

CSS

  • ✓ Адаптивный дизайн (медиа-запросы)
  • ✓ Проверка на разных размерах экрана
  • ✓ Правильное выравнивание элементов
  • ✓ Согласованные цвета и шрифты
  • ✓ Проверка на разных браузерах
  • ✓ Оптимизированные изображения

Доступность

  • ✓ Контрастность цветов (не менее 4.5:1)
  • ✓ Возможность навигации с клавиатуры
  • ✓ ARIA-атрибуты при необходимости
  • ✓ Читаемость для скринридеров

Производительность

  • ✓ Оптимизированные изображения
  • ✓ Минимизированные CSS и JS файлы
  • ✓ Использование сжатия (gzip/brotli)
  • ✓ Оптимизация загрузки шрифтов

SEO

  • ✓ Meta теги (title, description)
  • ✓ Правильная иерархия заголовков
  • ✓ Alt-атрибуты для изображений
  • ✓ Структурированные данные (Schema.org)
