Типы данных в JavaScript

Примитивные типы данных

  • Number — числа (целые и дробные): 42, 3.14
  • String — строки: "hello", 'world'
  • Boolean — логические значения: true, false
  • Undefined — значение не присвоено: undefined
  • Null — намеренно пустое значение: null
  • Symbol — уникальные идентификаторы (ES6+)
  • BigInt — целые числа произвольной длины (ES2020+)

Объектные типы данных

  • Object — коллекции пар ключ-значение
  • Array — упорядоченные коллекции данных
  • Function — функции (в JavaScript функции являются объектами)
  • Date — объекты даты и времени
  • RegExp — регулярные выражения
  • Error — объекты ошибок

Проверка типа данных

Для проверки типа данных используется оператор typeof:

typeof "hello" возвращает "string"

typeof 42 возвращает "number"

Для объектов и массивов typeof возвращает "object", поэтому для проверки массивов используется Array.isArray().
