HTML основы

Базовый обзор HTML

HTML (HyperText Markup Language) — это язык гипертекстовой разметки, стандартный для документов, предназначенных для отображения в браузере.

Структура HTML документа

Каждый HTML-документ имеет определенную структуру:

  • <!DOCTYPE html> — объявление типа документа
  • <html> — корневой элемент документа
  • <head> — метаданные документа
  • <body> — видимое содержимое страницы

Теги и атрибуты

HTML-теги — это элементы языка, определяющие структуру и содержание веб-страницы. Атрибуты предоставляют дополнительную информацию об элементе.

Семантическая разметка

Семантические теги описывают значение элементов для браузера и пользователей:

  • <header> — шапка страницы или раздела
  • <nav> — навигационные ссылки
  • <main> — основное содержимое
  • <section> — логический раздел
  • <article> — независимый контент
  • <aside> — сопутствующая информация
  • <footer> — нижний колонтитул
