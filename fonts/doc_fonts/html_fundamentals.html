Базовый обзор HTML

Основы HTML

  • Роль HTML : HTML представляет содержание и структуру веб-страницы.
  • Элементы HTMLЭлементы — это строительные блоки HTML-документа. Они представляют собой заголовки, абзацы, ссылки, изображения и многое другое. Большинство элементов HTML состоят из открывающего тега ( <elementName>) и закрывающего тега ( </elementName>).

Вот основной синтаксис:


  <elementName>Content goes here<elementName>
  • Элементы Void : Элементы Void не могут иметь никакого содержимого и имеют только начальный тег. Примеры включают элементы img и meta. 
    <img>
    <meta>

В некоторых кодовых базах часто встречается косая черта / внутри элемента void. Оба варианта допустимы:

    <img>
    <img>
  • Атрибуты : Атрибут — это значение, помещённое в открывающий тег HTML-элемента. Атрибуты предоставляют дополнительную информацию об элементе или определяют его поведение. Ниже представлен базовый синтаксис атрибута:

  <elementName attribute="value"><elementName>

Булев атрибут — это атрибут, который может присутствовать или отсутствовать в HTML-теге. Если присутствует, значение — true, в противном случае — false. Примеры булевых атрибутов: disabled, readonly и required.

  • Комментарии : Комментарии используются в программировании для того, чтобы оставлять заметки для себя и других разработчиков в коде. Вот синтаксис комментария в HTML:
<!--This is an HTML comment.-->
