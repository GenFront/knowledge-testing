Полная шпаргалка: "Div vs Семантика"

Когда использовать div, а когда семантические теги

Использовать div когда:

  • Нужна только группировка элементов для стилизации
  • Нет логического смысла в группировке
  • Это чисто визуальная обертка без содержательной роли
  • Создание вспомогательных контейнеров для позиционирования

Использовать семантические теги когда:

  • Элемент имеет логическую роль в структуре документа
  • Важна доступность и SEO
  • Контент имеет определенное назначение (навигация, статья, боковая панель и т.д.)
  • Повышается понятность кода для других разработчиков

Примеры правильного использования

Вместо: <div class="header"> → использовать <header>

Вместо: <div class="navigation"> → использовать <nav>

Вместо: <div class="article"> → использовать <article>
