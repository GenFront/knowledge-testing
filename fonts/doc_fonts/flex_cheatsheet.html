Интерактивная шпаргалка по CSS Flex

Быстро понять flex-grow, flex-shrink, flex-basis и их влияние на размеры

Краткая таблица

Свойство Что делает Пример
flex-grow Сколько долей свободного пространства получит элемент (если есть свободное место) flex: 2 1 0 → grow = 2
flex-shrink Во сколько раз элемент уменьшается при нехватке места (чем больше — тем сильнее сжимается) flex: 1 2 200px → shrink = 2
flex-basis Базовый размер элемента (основа, от которой считаются grow/shrink). auto — используется width/height/контент, 0 — игнорировать естественный размер flex-basis: 0 | auto | 200px

Контролы

Элемент A — flex: 1 1 auto
Элемент B — flex: 1 1 auto
Элемент C — flex: 1 1 auto

Мини-примеры (быстрая шпаргалка)

  1. Равные колонки: flex: 1 1 0 — игнорирует width, делит пространство поровну.
  2. С учётом width: flex: 1 1 auto; width: 200px; — базовый размер 200px, затем распределяется свободное место.
  3. Фиксированный элемент: flex: 0 0 150px; — элемент всегда 150px, не растёт и не сжимается.
Сделано как интерактивная шпаргалка — меняй параметры и наблюдай.
