Интерактивная шпаргалка по CSS Flex

Flexbox: основные свойства

Свойства flex-контейнера

  • display: flex — делает элемент flex-контейнером
  • flex-direction — направление основной оси (row, column, row-reverse, column-reverse)
  • justify-content — выравнивание по основной оси (flex-start, flex-end, center, space-between, space-around, space-evenly)
  • align-items — выравнивание по поперечной оси (stretch, flex-start, flex-end, center, baseline)
  • flex-wrap — перенос элементов (nowrap, wrap, wrap-reverse)
  • align-content — выравнивание строк при наличии свободного места (flex-start, flex-end, center, space-between, space-around, stretch)

Свойства flex-элементов

  • flex-grow — определяет, насколько элемент может расти относительно других
  • flex-shrink — определяет, насколько элемент может сжиматься относительно других
  • flex-basis — начальный размер элемента перед распределением свободного места
  • flex — сокращение для flex-grow, flex-shrink и flex-basis
  • align-self — выравнивание отдельного элемента (override для align-items)
  • order — порядок элемента среди других (по умолчанию 0)

Практические примеры

Центральное выравнивание: display: flex; justify-content: center; align-items: center;

Равномерное распределение: display: flex; justify-content: space-between;
