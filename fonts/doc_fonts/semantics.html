Семантические теги HTML

Семантические теги описывают значение содержимого для браузера и пользователей.

Структурные теги

  • <header> — шапка страницы или раздела
  • <nav> — навигационные ссылки
  • <main> — основное содержимое
  • <section> — логический раздел
  • <article> — независимый контент
  • <aside> — сопутствующая информация
  • <footer> — нижний колонтитул

Польза семантики

  • Улучшает доступность для пользователей с ограниченными возможностями
  • Повышает SEO (поисковую оптимизацию)
  • Облегчает понимание структуры документа
  • Упрощает поддержку и развитие кода
