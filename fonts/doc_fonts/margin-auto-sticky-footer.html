margin: 0 auto; и sticky footer

Выравнивание по центру с помощью margin: 0 auto

Свойство margin: 0 auto; используется для центрирования блочных элементов по горизонтали.

Как работает

  • margin: 0 auto; устанавливает верхнее и нижнее поле равным 0
  • Левое и правое поле устанавливаются равными и автоматически распределяются
  • Элемент центрируется в пределах своего родительского контейнера
  • Работает только с блочными элементами, имеющими определенную ширину

Sticky footer

Sticky footer — это техника, при которой футер всегда остается внизу окна просмотра, даже если контент короткий.

Способы реализации sticky footer

  • Flexbox: установка display: flex и flex-direction: column для body, затем flex: 1 для main
  • CSS Grid: использование grid-template-rows: auto 1fr auto
  • Calc(): использование calc(100vh - header_height - footer_height)
