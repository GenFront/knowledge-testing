# Knowledge Testing - Тестирование знаний

📚 **Проект для интерактивного тестирования знаний по веб-разработке**

## 🚀 Функционал

- Таблица-оглавление с отслеживанием прогресса (✅/🔄)
- Интерактивные тесты с мгновенной проверкой
- Поддержка темной/светлой темы
- Аудио-эффекты (опционально)

# 🚀 Мой проект с системой напоминаний

Проект включает в себя умную систему напоминаний перед запуском приложения.

## 📁 Структура проекта

project/
├── index.html # Авто-редирект на напоминание
├── reminder.html # Страница с напоминанием
├── app.html # Основное приложение
├── reminder.css # Стили для напоминания
├── style.css # Стили основного приложения
└── scripts/
└── reminder.js # Логика напоминания


## 🎯 Как это работает

1. **Запуск проекта** → открывается `index.html`
2. **Авто-редирект** → переход на `reminder.html`
3. **Напоминание** → показывается важная информация
4. **Таймер** → через 5 секунд автоматический переход к приложению
5. **Основное приложение** → загружается `app.html`

## 🛠 Установка и запуск

### Локальный запуск:
```bash
# Клонируйте репозиторий

npx http-server
