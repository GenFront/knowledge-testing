# Knowledge Testing - Тестирование знаний

📚 **Проект для интерактивного тестирования знаний по веб-разработке**

## 🚀 Функционал

- Таблица-оглавление с отслеживанием прогресса (✅/🔄)
- Интерактивные тесты с мгновенной проверкой
- Поддержка темной/светлой темы
- Аудио-эффекты (опционально)

## 📂 Структура проекта

```bash
knowledge-testing/
├── images/               # Графические ресурсы
│   ├── icons/            # Иконки (статусы, кнопки)

├── styles/               # Стилизация

├── tests/                # Тесты
│   ├── test-html.html    # Тест по HTML
│   └── test-css.html     # Тест по CSS
├── audio/                # Звуковые эффекты

🛠 Установка

Клонировать репозиторий:

git clone https://github.com/ваш-логин/knowledge-testing.git

Открыть index.html в браузере

🎮 Использование

Выбрать тему в таблице
Пройти тест (пример: tests/test-html.html)
Просмотреть результаты:
Правильные ответы подсвечиваются зеленым
Прогресс сохраняется в LocalStorage
```
