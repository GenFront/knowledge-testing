# Knowledge Testing - Тестирование знаний

📚 **Проект для интерактивного тестирования знаний по веб-разработке**

## 🚀 Функционал

- Таблица-оглавление с отслеживанием прогресса (✅/🔄)
- Интерактивные тесты с мгновенной проверкой
- Поддержка темной/светлой темы
- Аудио-эффекты (опционально)

## 📂 Структура проекта

```bash
knowledge-testing/
├── images/               # Графические ресурсы
│   ├── icons/            # Иконки (статусы, кнопки)
│   └── backgrounds/      # Фоновые изображения
├── styles/               # Стилизация
│   ├── main.css          # Базовые стили
│   └── dark-theme.css    # Темная тема
├── scripts/              # Логика
│   └── app.js            # Основной скрипт
├── tests/                # Тесты
│   ├── test-html.html    # Тест по HTML
│   └── test-css.html     # Тест по CSS
├── audio/                # Звуковые эффекты
│   ├── click.mp3         # Клики
│   └── complete.wav      # Завершение теста
├── docs/                 # Документация
│   └── конспекты.md      # Учебные материалы
├── vendor/               # Внешние библиотеки
│   └── jquery.min.js     # jQuery (пример)
└── index.html            # Главная страница


🛠 Установка

Клонировать репозиторий:

git clone https://github.com/ваш-логин/knowledge-testing.git

Открыть index.html в браузере

🎮 Использование

Выбрать тему в таблице
Пройти тест (пример: tests/test-html.html)
Просмотреть результаты:
Правильные ответы подсвечиваются зеленым
Прогресс сохраняется в LocalStorage

📌 Особенности

Аудио-эффекты: Включены в папке /audio (можно отключить в настройках)
Темная тема: Активируется через dark-theme.css
Конспекты: Доступны в /docs в формате Markdown
```

1. Как скринридеры читают Unicode-символы

Для ваших символов с HTML-кодами:

&#x2705; (✅) → Читается как "белая галочка в зелёном квадрате" (зависит от скринридера)
&#x1F504; (🔄) → "круговые стрелки против часовой стрелки"
&#10060; (❌) → "крестик"
⏳ &#x23F3; В планах
Проблема: Некоторые скринридеры могут произносить коды буквально ("амперсанд х два семь ноль пять").
